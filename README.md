# Valkey Inspector

A web UI for checking what keys are available in a Valkey instance or cluster.

Built by Human and AI (Gemini Code Assist) using the following stack:

- Gemini Code Assist (Free tier)
- VSCode
- Ubuntu
- Golang 1.25
- Valkey
- Docker and Docker compose
- Frontend
    - pnpm
    - Vue and Nuxt
    - Tailwindcss
